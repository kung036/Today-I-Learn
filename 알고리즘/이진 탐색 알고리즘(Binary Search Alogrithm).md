## **이진 탐색 알고리즘(Binary Search Alogrithm)**

-   `이진 탐색 알고리즘(Binary Search Algorithm)` : 정렬된 데이터에서 절반씩 나누어서 분할 정복기법으로 탐색하는 알고리즘
-   한계
    - 배열을 통해서만 구현 가능 
    - 배열을 정렬해야 이진 탐색 알고리즘 적용 가능
    <br></br>
-   시간 복잡도 : O(log N)
-   사용되는 경우
    1.  사전에서 단어 검색
    2.  도서관에서 도서 검색
    3.  대규모 시스템의 리소스 사용 현황 파악
    4.  반도체 테스트 프로그램의 디지털 & 아날로그 레벨 측정
    <br></br>
